# 🍱 Food Waste Reduction Portal

## 📌 Overview

The **Food Waste Reduction Portal** is a web-based application designed to reduce food wastage by connecting food donors with people in need. This platform allows users to donate surplus food and helps receivers access available food efficiently.

## 🚀 Features

* 👤 User Registration and Login
* 🍲 Add Food Donations
* ✏️ Update and Delete Food Details (CRUD Operations)
* 📋 View Available Food Listings
* 🔗 Database Connectivity using JDBC
* ⚙️ Backend processing using Java Servlets

## 🛠️ Technologies Used

Frontend:HTML, CSS
Backend:Java (Servlets)
Database:MySQL
Connectivity:JDBC

## 🎯 Objective

To minimize food wastage and help needy people by creating a platform that connects food donors with receivers in an efficient way.

## ⚙️ Setup Instructions

1. Clone the repository

   ```
   git clone https://github.com/your-username/food-waste-portal.git
   ```

2. Import project into Eclipse (Dynamic Web Project)

3. Configure Apache Tomcat Server

4. Set up MySQL Database:

   * Create database
   * Run `schema.sql`

5. Update database credentials in JDBC code

6. Run the project on server
